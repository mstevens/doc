# Install it locally

You need [yarn](https://www.yarnpkg.com/en/docs/install).

```
yarn install
yarn honkit serve
```

At the end, you should see :

```
Starting server ...
Serving book on http://localhost:4000
```

Update file locally and content will be live-reloaded (might take few seconds).
